# AGROX PROJECT ROADMAP
**AgroX Organic Exports — Digital Ecosystem Development Plan**
Last Updated: June 2026

## Overview

This roadmap documents the phased development plan for the AgroX digital platform — transforming a standard export company website into a world-class, content-rich B2B trade intelligence portal capable of ranking for thousands of commercial agricultural keywords globally.

The architecture is designed to scale from a core 20-page site (Phase 1–2) to a 1,000+ page authority platform (Phase 8–10) without requiring structural refactoring.

## Technical Standards (All Phases)

| Requirement | Standard |
|---|---|
| Base Theme | True off-black `#050505`, Gold accent `#d4a017` |
| Display Font | Bebas Neue (all uppercase, letter-spacing 1.5px) |
| Body Font | Poppins / Plus Jakarta Sans (weight 300–500) |
| Motion Library | GSAP + Lenis Smooth Scroll |
| Parallax | CSS custom properties + `requestAnimationFrame` |
| Button States | `transform: scale(0.97)` on `:active` |
| Glass Components | `rgba(20,20,20,0.6)` + `backdrop-filter: blur(15px)` |
| PHP Include | All pages use `header.php` + `footer.php` |
| Path Handling | `$path` smart detection for root vs subdirectory |
| Mobile | Fully responsive, hamburger accordion mobile menu |
